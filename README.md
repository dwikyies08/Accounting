# Hal hal yang harus dilakukan saat clone project ini
1. Clone Repo
```sh
   git clone https://github.com/zackyelsegaf/ptanugerahmagellonusantara.git
   ```
2. Install Composer
```sh
   composer install
   ```
3. Install NPM
```sh
   npm install
   ```
4. Migrate Database
```sh
   php artisan migrate
   ```
5. Registrasi User Baru
```sh
   http://127.0.0.1:8000/register
   ```
